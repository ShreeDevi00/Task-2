1.Differnce between block level and inline elements in html.
ANSWER: Block level elements
        *start on a new line
		*take full width of the page
		*can contain inlineand other block element
		*for example:<div>,<p>,<h>.
	* inline elements
	*do not start on anew line
	*take only requried width
	*usually containstex or other inline elements
	*For example:<span>,<a>,<strong>
